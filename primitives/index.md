Spandrel is built on two primitives and a governance layer. The simplicity is intentional — uniform primitives make the system legible at any scale.

- [Things](/primitives/things) — the atomic unit of knowledge
- [Collections](/primitives/collections) — Things that contain other Things
- [Tags](/primitives/tags) — cross-cutting discovery across the tree
- [Governance](/primitives/governance) — the access control overlay
