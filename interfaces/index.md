Spandrel provides multiple interfaces, all backed by the same data through GraphQL.

- [GraphQL](/interfaces/graphql) — the universal interface layer (the invariant)
- [MCP](/interfaces/mcp) — 8 tools for agent and LLM access
- [CLI](/interfaces/cli) — developer commands for compile, dev, and MCP mode
