Three primary actor types interact with Spandrel in different ways:

- [Context Engineer](/user-journeys/context-engineer) — builds and maintains the system, works with files locally
- [Analyst](/user-journeys/analyst) — explores and queries the graph in depth
- [MCP Consumer](/user-journeys/mcp-consumer) — navigates the graph via MCP tools from any client
